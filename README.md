# OpenCompliance Site

This directory is the source of truth for the public `opencompliance-foundation/site` repository.

It is a standalone static bundle intended for GitHub Pages under `opencompliance.aguilar-pelaez.co.uk`.

## What belongs here

- the public explanation of OpenCompliance,
- the trust model and publication model,
- the roadmap and foundation pages,
- and only static assets that are safe to publish openly.

## What does not belong here

- private customer material,
- unpublished implementation notes,
- credentials,
- or claims that outrun what the public repos actually ship.
